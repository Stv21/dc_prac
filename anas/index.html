Distributed Computing Files

  1. 1. IPC using Message Passing - Publisher-subscriber using ThreadPoolExecutor and Queue for inter-process communication.
  2. 2. RPC (Remote Procedure Call) - XML-RPC server exposes add() function; client calls it remotely over localhost.
  3. 3. Bully Algorithm - Election algorithm — nodes delegate upward until highest available ID becomes coordinator.
  4. 4. Ring Algorithm - Election via ring topology — message travels full circle, max ID declared leader.
  5. 5. Ricart-Agarwala Algorithm - Distributed mutual exclusion using Lamport clocks, threading, and REQUEST/ACK/RELEASE messages.
  6. 6. Deadlock Detection (Centralized) - Centralized approach — detects which process is blocked waiting for a resource held by another.
  7. 7. Random Load Balancing - Each incoming request is assigned to a randomly chosen server.
  8. 8. Round Robin Load Balancing - Requests distributed in cyclic order across all servers using an index pointer.
  9. 9. Least Connection Load Balancing - Request routed to the server currently handling the fewest active connections.
  10. All Code Combined - All 9 experiments in one page
